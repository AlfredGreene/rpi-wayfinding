rpi-wayfinding
==============

PHP application for wayfinding using a Raspberry Pi and an HDMI monitor.

Requirements
-------------------------

 * PHP 5.3+
 * GD Library
 * feh (for RPi)
 * Puppet master (optional)
 * cURL with NTLM support (7.23.0+ recommended)
 * Exchange 2007 or 2010*
